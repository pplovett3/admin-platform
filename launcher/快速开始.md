# YF课程启动器 - 快速开始

## 5分钟快速上手

### 1. 启动后端服务器

```bash
cd admin-platform/server
npm start
```

确保服务器运行在 `http://localhost:4000`。

### 2. 配置课程

编辑 `launcher/courses.json`：

```json
{
  "courses": [
    {
      "courseId": "你的课程ID",
      "appPath": "C:/Windows/System32/notepad.exe",
      "name": "测试课程"
    }
  ]
}
```

**获取课程ID：** 登录管理平台 → 课程管理 → 查看课程详情

### 3. 启动启动器

```bash
cd admin-platform/launcher
npm run dev
```

### 4. 测试登录

**测试账号：**
- 手机号：13800000000（或你创建的测试账号）
- 密码：admin123（或对应密码）

### 5. 激活课程

在管理平台：
1. 进入"激活码管理"
2. 生成激活码（选择课程）
3. 用测试学生账号登录
4. 在学生端激活课程

### 6. 启动课程

在启动器中：
1. 登录
2. 查看课程列表
3. 点击"启动课程"
4. 观察应用是否成功启动
